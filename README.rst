fixcors plugin for `Tutor <https://docs.tutor.edly.io>`__
#########################################################

adjust caddy cors for tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/QueriumCorp/tutor-contrib-fixcors

Usage
*****

.. code-block:: bash

    tutor plugins enable fixcors
